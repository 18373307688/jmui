Features
--------

* Design for mobile
* An enterprise-class design language and high quality UI style
* Include IOS and Material Themes base on Framework7(https://github.com/nolimits4web/Framework7)
* Rich library of mobile UI components
  * view
  * page
  * navbar
  * toolbar
  * tabbar
  * button
  * card
  * list
  * notification
  * modal
  * block
  * form
  * preloader
  * ...

Requirements
------------

* node `^4.3.0`
* npm `^3.0.0`
